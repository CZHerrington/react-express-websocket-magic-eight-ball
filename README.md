# React/Express/Websocket Magic Eight Ball SPA


to deploy:
(in project root directory)
1. `npm start`

to develop:
(in project root directory)
1. `npm run build`
2. `npm run dev`
3. `npm start` (**in new terminal window**)

navigate to `http://localhost`:`$PORT`, where $PORT is specified in the `.env` file (or 8000 if unspecified)
